# Project Documentation

This folder contains contributor-facing documentation that is more detailed than the main `README.md`.

## Files

- `manual_acceptance.md` - manual QA checklist for validating the app after changes.
- `llm_wiki_workflow.md` - workflow for creating and updating the `llm_wiki.db` database that powers the app.

The app does not require this folder at runtime. These documents are for maintainers, contributors, and users who want to understand or validate the project.
